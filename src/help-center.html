Sab categories ke common sawaal

GigLega kya hai aur kaise kaam karta hai?
+

GigLega Gurugram ka #1 hyperlocal gig platform hai jahan local workers aur clients directly connect hote hain — bina kisi middleman ke.

  • Workers available gigs browse karte hain aur apply karte hain
  • Clients job post karte hain aur suitable workers dhundte hain
  • Dono ke liye platform 100% free hai abhi Beta phase mein
Pro tip: Register karo → Profile complete karo → Browse/Post karo → Connect ho jao. Bas itna simple hai!
Kya yeh helpful tha?
Account kaise banate hain?
+

Account banana 1 minute se kam mein ho jaata hai:

  • signup.html par jao
  • Apna naam, email, phone aur password dalo
  • Role choose karo — Worker (kaam dhundna) ya Client (kaam karwana)
  • "Create Free Account" button press karo
Note: Abhi email verification nahi hai Beta mein — direct access milta hai register karte hi.
Kya yeh helpful tha?
GigLega free hai kya — koi hidden charges?
+

Haan, bilkul free hai! Koi hidden charges nahi hain:

  • Workers ke liye register karna — Free ✅
  • Gigs browse aur apply karna — Free ✅
  • Clients ke liye gig post karna — Free ✅ (Beta)
  • Platform use karna — Free ✅

GigLega platform fees ya commission nahi leta. Payment seedha worker aur client ke beech hoti hai.

Future mein: Premium features aane par optional paid plans introduce ho sakte hain — lekin basic features free rahenge.

Gig ke liye apply kaise karein?
+
  1. Browse Gigs page par jao
  2. Apni category aur location filter karo
  3. Interesting gig pe click karo — detail page khulega
  4. "Apply Now" button press karo
  5. Apna naam, phone aur short message dalo
  6. Submit karo — client ko notification jaayegi
Better chances ke liye: Message field mein apna relevant experience briefly mention karo — generic "apply kar raha hoon" se zyada effective hota hai.
Kya yeh helpful tha?
Apply karne ke baad kya hoga?
+

Apply karne ke baad:

  • Aapki application client ke dashboard mein dikhai deti hai
  • Client aapko directly phone/WhatsApp karta hai — jo number diya hai usi par
  • Urgent gigs mein usually 1-24 ghante mein contact hota hai
  • Agar koi response na aaye toh aap gig detail page par jaake client se directly contact kar sakte ho

Dashboard mein "My Applications" section mein apni saari applications track kar sakte ho.

Kya yeh helpful tha?
Profile complete karna kyun zarooori hai?
+

Complete profile se aapke chances significantly improve hote hain kyunki:

  • Clients pehle complete profiles wale workers pe trust karte hain
  • Skills mention karne se relevant gigs zyada milte hain
  • Bio se client samajhta hai ki tum suitable candidate ho
  • Location se local gigs matching improve hoti hai

Profile mein yeh fill karo: Name, Phone, Location, Bio, Skills — 100% completion ke liye.

Quick tip: Dashboard → Edit Profile mein jao aur 2 minute mein complete karo.
Kya yeh helpful tha?
Ek hi gig par baar baar apply kar sakte hain?
+

Technically aap multiple baar apply kar sakte ho lekin yeh recommended nahi hai:

  • Client ko spam lagta hai — reject karne ke chances badh jaate hain
  • Ek thoughtful application zyada effective hai
  • Agar pehli application mein kuch choot gayi ho toh contact form se update bhej sakte ho
Better approach: Ek strong message likho pehli baar mein — experience, availability aur contact clearly mention karo.
Kya yeh helpful tha?
Koi gig fake lage toh kya karein?
+

Agar koi gig suspicious lage — advance fee maange, unrealistic salary ho, ya personal financial info maange — toh:

  • Apply mat karo
  • Gig detail page par "Report" button use karo
  • Humein WhatsApp karo: +91 93196 35257
  • Screenshot ke saath report karo
Red flags: Advance payment maangna, interview fee, unrealistic salaries (₹5000/day for basic work), vague job descriptions.
Kya yeh helpful tha?
Gig post kaise karein?
+
  1. Login karo (account nahi hai toh pehle banao)
  2. Post a Gig page par jao
  3. Title, category, description fill karo
  4. Budget aur type (per hour/day/project) set karo
  5. Location aur urgency set karo
  6. "Post Gig" press karo — turant live ho jaayega
Better responses ke liye: Clear description likho — exactly kya kaam hai, kab chahiye, aur payment kab milega. Zyada detail = better applicants.
Kya yeh helpful tha?
Applications kahan dikhti hain?
+

Applications manage karne ke 2 jagah hain:

  • Dashboard → My Posted Gigs: Har gig pe click karo — uski saari applications expand ho jaayengi
  • Dashboard → All Applications: Flat list mein saari incoming applications ek saath

Har applicant ka naam, phone aur message dikh ta hai. Contact button se directly WhatsApp karo.

Kya yeh helpful tha?
Gig close ya edit kaise karein?
+

Gig close karna:

  • Dashboard → My Posted Gigs par jao
  • Gig ke "Close Gig" button press karo
  • Closed gig browse page mein nahi dikhegi — workers apply nahi kar sakte
  • "Reopen" button se wapas active kar sakte ho
Kaam ho gaya? Gig hire ho jane ke baad close karo taaki extra applications na aayein.

Beta phase mein koi strict limit nahi hai — aap multiple gigs post kar sakte ho.

Lekin recommend karte hain ki:

  • Sirf genuine, active requirements post karo
  • Purani ya filled positions ko close karo
  • Spam posting account suspension ka reason ban sakti hai

Payment kaise hoti hai?
+

GigLega payment process nahi karta — yeh ek connection platform hai.

Payment client aur worker ke beech directly hoti hai:

  • 💵 Cash (in hand)
  • 📱 UPI (PhonePe, GPay, Paytm)
  • 🏦 Bank Transfer / NEFT
  • Jo bhi dono parties agree karein
Safe rahne ke liye: Kaam start karne se pehle payment terms — amount, timing, mode — clearly agree kar lo. Writing mein (WhatsApp pe) confirm karo.
Kya yeh helpful tha?
Client ne payment nahi ki — kya karein?
+

Yeh unfortunate situation mein:

  • Pehle client se directly contact karo — politely remind karo
  • WhatsApp messages ko evidence ke roop mein save karo
  • GigLega ko report karo — contact form ya WhatsApp
  • Hum mediation karenge lekin legally enforce nahi kar sakte
  • Agar amount significant hai — consumer court ya local police se help lo
Prevention best: Agar possible ho toh advance ya partial payment pehle lo. Small tasks ke liye kaam pura hone ke baad turant payment lo.
Kya yeh helpful tha?

Account delete kaise karein?
+

Account delete karne ke 2 ways hain:

Option 1 — Self delete (Beta):

  • Browser DevTools kholna (F12)
  • Application → LocalStorage → giglega.in → Clear All
  • Yeh saara data instantly delete kar deta hai

Option 2 — Request karo:

  • Email: Giglega.official@gmail.com
  • Subject: "Account Deletion Request"
  • Hum 7 business days mein process karenge

Data ek device se dusre device par kyun nahi dikh raha?
+

Yeh Beta phase ki limitation hai — GigLega abhi localStorage use karta hai.

Matlab aapka data sirf us device aur browser mein save hota hai jahan register kiya.

  • Mobile par kiya tha → Desktop par nahi dikhega
  • Chrome mein kiya tha → Firefox mein nahi dikhega
Solution coming soon: Backend server launch hone par cross-device sync automatically kaam karega — sab devices par ek account!
Kya yeh helpful tha?
Page load nahi ho raha / website broken dikh rahi hai
+

Yeh steps try karo order mein:

  1. Page hard refresh karo — Ctrl+Shift+R (Windows) ya Cmd+Shift+R (Mac)
  2. Browser cache clear karo
  3. Different browser mein try karo
  4. Internet connection check karo
  5. Incognito/private mode mein try karo

Agar phir bhi issue ho toh screenshot ke saath WhatsApp karo — hum fix karenge!
