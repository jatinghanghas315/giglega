4 Data Storage

Beta Phase: Abhi GigLega ek frontend-only platform hai. Saara data aapke browser ke localStorage mein store hota hai.

Iska matlab:

  • Data sirf aapke device par save hota hai
  • Browser clear karne se data delete ho sakta hai
  • Dusre device par data nahi hoga (sync nahi hota)
  • Hum remotely aapka data access nahi kar sakte (Beta mein)

Future (Production phase) mein: Jab backend launch hoga toh data secure encrypted servers par store hoga — India mein hosted, data residency laws ke compliant.

Data retention: Active accounts ka data tab tak rakha jaata hai jab tak account active hai. Inactive accounts (12+ months) ka data delete kiya ja sakta hai — advance notice ke saath.

5 Data Sharing

Hum aapka personal data sell nahi karte — kabhi nahi.

Sirf inhi cases mein data share hota hai:

  • Platform functionality: Jab aap gig apply karte ho toh client ko aapka name, phone aur message dikhta hai — yeh platform ka core function hai
  • Legal requirement: Court order ya government request par legally required information share karna pad sakta hai
  • Safety: Fraud ya illegal activity rokne ke liye authorities ko report karna
  • Business transfer: Agar GigLega acquire ya merge ho toh — users ko pehle notify kiya jayega
⚠️ Jab aap gig apply karte ho toh aapka contact info client ko visible hota hai — yeh intentional hai taaki wo aapko hire kar sakein. Isliye accurate info dalo.

9 Data Security

Hum aapka data protect karne ke liye yeh measures lete hain:

  • HTTPS — saara platform encrypted connection par serve hota hai
  • Passwords hashed store kiye jaate hain (future backend mein bcrypt)
  • Session tokens expiry ke saath issue kiye jaate hain
  • Suspicious activity monitor ki jaati hai
  • Regular security reviews

Lekin 100% security guarantee possible nahi hai — internet par koi bhi system completely breach-proof nahi hota. Agar aapko koi security issue mile toh turant report karo.

Beta phase mein: Data localStorage mein hai — device lock rakhna, public computers par logout karna aapki responsibility hai.
